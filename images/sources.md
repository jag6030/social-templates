## How to Use This File
- Add an entry every time you add an image to the repo
- Paste the source URL even if it's a stock site — you'll never remember later
- License types: **Own Photo** | **Unsplash (free)** | **Pexels (free)** | **Paid Stock** | **Canva** | **Public Domain**
- One entry per image file

## Naming Convention
`subject-keyword_town_YYYY-MM.jpg`

Examples:
- `link-trail_haddon-heights_2026-03.jpg`
- `golf-lane_voorhees_2026-03.jpg`
- `farmers-market_moorestown_2026-04.jpg`

Multiple images for one post — add a letter suffix:
- `link-trail_haddon-heights_2026-03a.jpg`
- `link-trail_haddon-heights_2026-03b.jpg`
- `link-trail_haddon-heights_2026-03c.jpg`
